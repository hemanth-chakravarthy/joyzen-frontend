# Joyzen — Redefining Healthcare

### Designed for trust. Built for healthcare.
[Live Demo on Vercel](https://joyzen-frontend.vercel.app/)

Joyzen is a premium, high-performance landing page built with React. It features a state-of-the-art "Liquid Glass" UI, seamless responsiveness, and next-gen performance optimizations.

---

## Key Features

- **Liquid Glass UI**: A bespoke design system utilizing deep backdrop blurs, subtle highlights, and high-shine glass effects.
- **Universal Responsiveness**: Pixel-perfect experiences tailored for:
  - **Mobile**: Animated dropdown navigation and vertical content stacking.
  - **Tablet and iPad**: Optimized layouts for mid-sized screens.
  - **4K Displays**: High-definition scaling for 2560px+ monitors.
- **Next-Gen Performance**:
  - **WebP Assets**: All photography and gradients converted to next-gen formats for minimum bandwidth.
  - **Lazy Loading**: Intelligent asset delivery ensures instant initial page loads.
  - **Smooth Motion**: Powered by Framer Motion for fluid, organic transitions.
- **Interactive UX**: Custom hamburger menu, high-contrast hover states, and smooth-scroll anchor links.

---

## Tech Stack

- **Core**: React (JavaScript)
- **Styling**: Tailwind CSS and Vanilla CSS
- **Animation**: Framer Motion
- **Typography**: Satoshi and Epilogue

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone [repository-url]
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch development server**:
   ```bash
   npm start
   ```

4. **Production Build**:
   ```bash
   npm run build
   ```
